# C++ 设计模式


## Creational Patterns(创建型模式)

| 重要程度 | 英文名称 | 中文名称 | 简述 |
| :---: | :------ | :------ | :------------ |
| *** | Factory Method | 工厂方法模式 | 在父类中提供一个创建对象的方法, 允许子类决定实例化对象的类型 |
| *** | Abstract Factory | 抽象工厂模式 | 能创建一系列相关的对象， 而无需指定其具体类 |
| *** | Builder | 生成器模式 |  |
| **  | Prototype | 原型模式 |  |
| **  | Singleton | 单例模式 |  |
